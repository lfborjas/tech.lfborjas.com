  • Las provisiones están fuera de banda

  • Totalitarismo accidental: perdemos el propósito de la web al tener pereza de entender nuestros protocolos, estándares y medios

La solución: hipermedios

Usando, por ejemplo el tipo de hipermedios HAL

          
/* GET http://api.zaphod.net/ */
{
  "_links": {
    "http://api.zaphod.net/rels/login": {"href": "/accounts/login"},
    "http://api.zaphod.net/rels/account": {"href": "/accounts/{id}", "templated": true}
  }
}
/* POST message._links.login */
{
  "authentication_token": xyz42,
  "_embedded": {
    "account": {
        {"_links": {"self": {"href": "/accounts/42"}}}
        "name": "Zaphod Beeblebrox",
        "id": 42
    }
  },
  "_links": { "messages": "/accounts/42/messages" }
}
  • Definimos el tipo de medios ("HAL")
  • Definimos las "link relations" (rels)
  • Ahora nuestros mensajes tienen provisiones
